## Get Started 
For a Chromium-based browser (e.g. Brave, Google Chrome), you can [download from the Chrome Web Store](https://chromewebstore.google.com/detail/filtered/bhcdiggpokngbonamlfgihnkjdeepmdg).

Or, manually:
1.  Clone or [download](https://github.com/galib-i/filtered/releases/latest) (and unzip) this repository.
2.  Open your browser's extensions page (`chrome://extensions/`).
3.  Toggle the "Developer mode" switch in the top right corner.
4.  Click "Load unpacked" and select this project's folder.
